Filter the event set by moderation status, then pick one item to edit. Click apply to update the working copy, then save changes. With an admin token the shared backend updates; otherwise the changes stay in this browser only.

Import from flyer or post

Paste a posting URL, direct image URL, or upload a flyer image. The backend will try page metadata plus OCR, then turn the result into an editable event candidate.

If the post URL exposes an image through og:image you can leave image URL blank. Upload is useful when the flyer is only on your computer.

Imported candidates are added to the working list first. Review and save when ready.
