**Goal:** Generate one static, individually-rankable HTML page per local business in the directory, in both English and Spanish, with full `LocalBusiness` structured data. Manual fetch script writes a snapshot of Google Places data; build script consumes the snapshot and produces ~50-80 new bilingual pages.

**Architecture:** A Node script (`scripts/fetch-businesses.js`) calls Google Places API New (REST) for each category around the town center, merges results, dedupes by `place_id`, applies a hardcoded block list, and writes `content/businesses-snapshot.json`. Each page emits `LocalBusiness` JSON-LD schema. The existing live `/businesses` directory page is updated to link to per-business pages. Sitemap gets all new URLs.

**Tech Stack:** Same as PRs 1-3 — Node 20+, vanilla JS, no runtime deps. Adds Google Places API New REST endpoint (server-side fetch). API key via `GOOGLE_PLACES_API_KEY` env var (with documented fallback).

## Locked design decisions

1. **API:** Google Places API New (`places.googleapis.com/v1/places:searchNearby`) — modern REST, supports the field-mask caching policy.
2. **API key:** Read `GOOGLE_PLACES_API_KEY` from environment. If unset, log a clear error and exit with code 2. The existing public client-side key (`AIzaSyANDAxggsq6pL3Jdgav2PxIk4KvoQmu8m4`) can be reused — Google Places API key permissions allow the same key for both browser and server use, but a server-restricted key is preferred. Document this in the script.
3. **Slug format:** `slugify(displayName) + '-' + place_id.slice(-6)` — guarantees uniqueness even with name collisions, keeps URLs short and readable. e.g. "Café Azul Cobalto" + place_id ending `bcd123` → `cafe-azul-cobalto-bcd123`.
4. **Categories fetched:** Same set the existing `businesses.html` queries — restaurant/meal_takeaway/meal_delivery, bar, cafe, store/grocery_or_supermarket/pharmacy/bakery, park/campground/tourist_attraction. Plus `lodging` (hotels) for tourist intent.
5. **Snapshot scope:** Up to 20 results per category type (Places API max), then merged + deduped by `place_id`. Expected ~50-80 unique businesses for a town this size.
6. **Block list:** Hardcoded array in `scripts/fetch-businesses.js` (currently just `PUNTO FIT`). PR 4b will optionally pull from the existing Upstash list.
7. **Snapshot is committed:** `content/businesses-snapshot.json` lives in git so the build is reproducible across machines and PRs. Refresh = re-run script + commit.
8. **Per-business URLs:**
   - English: `/en/businesses/[slug]`
   - Spanish: `/es/negocios/[slug]`
9. **Per-business page contents (v1):** Hero (name, primary type label, rating + review count), address (with map link), phone (click-to-call if present), opening hours table, "Open in Google Maps" CTA, breadcrumbs (Home > Directory > Name), `LocalBusiness` JSON-LD schema. NO photos in v1.
10. **Hreflang:** Each per-business page emits hreflang to its EN/ES counterpart and `x-default`.

```javascript
export function slugify(input) {
  if (!input) return '';
  return input
    .normalize('NFD')                        // split base char + combining diacritic
    .replace(/[̀-ͯ]/g, '')         // drop diacritics
    .toLowerCase()
    .replace(/[^a-z0-9\s-]/g, '')            // keep only letters, digits, spaces, dashes
    .replace(/[\s-]+/g, '-')                 // collapse runs of spaces/dashes to one dash
    .replace(/^-+|-+$/g, '');                // trim leading/trailing dashes
}

export function businessSlug(name, placeId) {
  const base = slugify(name) || 'place';
  const idTail = (placeId || '').slice(-6).toLowerCase();
  return idTail ? `${base}-${idTail}` : base;
}
```

function pickTypeLabel(business, typeLabels, lang) {
  const candidates = [business.primaryType, ...(business.types || [])];
  for (const t of candidates) {
    if (typeLabels[t] && typeLabels[t][lang]) return typeLabels[t][lang];

function ratingLine(business, labels, lang) {
  if (!business.rating) return '';
  const stars = '★'.repeat(Math.round(business.rating));
  const reviewsLabel = labels.reviews[lang];
  const count = business.userRatingCount ?? 0;
  return `<span class="stars">${stars}</span> ${business.rating.toFixed(1)} · ${count} ${reviewsLabel}`;
}

function phoneLine(business, labels, lang) {
  if (!business.internationalPhoneNumber) return labels.no_phone[lang];
  const safe = business.internationalPhoneNumber.replace(/\s+/g, '');
  return `<a href="tel:${safe}">${business.internationalPhoneNumber}</a>`;
}

function formatTime(hour, minute) {
  // returns 12h "5:00 PM" style — language-agnostic for our purposes
  const h12 = hour % 12 === 0 ? 12 : hour % 12;
  const ampm = hour < 12 ? 'AM' : 'PM';
  const m = String(minute ?? 0).padStart(2, '0');
  return `${h12}:${m} ${ampm}`;
}

function hoursHtml(business, weekdayLabels, labels, lang) {
  const periods = business.regularOpeningHours?.periods;
  if (!periods || periods.length === 0) {
    return `<p>${labels.no_hours[lang]}</p>`;
  }
  // Group periods by open.day
  const byDay = new Map(); // day → [{open, close}]
  for (const p of periods) {
    if (!p.open) continue;
    const day = p.open.day;
    if (!byDay.has(day)) byDay.set(day, []);
    byDay.get(day).push(p);
  }
  const rows = [];
  for (let d = 0; d < 7; d++) {
    const dayLabel = weekdayLabels[String(d)][lang];
    const ranges = byDay.get(d);
    if (!ranges || ranges.length === 0) {
      rows.push(`<dt>${dayLabel}</dt><dd>${labels.closed[lang]}</dd>`);
    } else {
      const text = ranges.map(r => {
        const opens = formatTime(r.open.hour, r.open.minute);
        const closes = r.close ? formatTime(r.close.hour, r.close.minute) : '?';
        return `${opens} – ${closes}`;
      }).join(', ');
      rows.push(`<dt>${dayLabel}</dt><dd>${text}</dd>`);
    }
  }
  return `<dl class="biz-hours">${rows.join('')}</dl>`;
}

    '@id': url,
    url,
    name: business.displayName,
    address: {
      '@type': 'PostalAddress',
      streetAddress: business.formattedAddress?.split(',')[0] ?? '',
      addressLocality: 'San José de Gracia',
      addressRegion: 'Jalisco',
      addressCountry: 'MX',
    },

  if (business.location) {
    schema.geo = {
      '@type': 'GeoCoordinates',
      latitude: business.location.latitude,
      longitude: business.location.longitude,
    };
  }
  if (business.internationalPhoneNumber) {
    schema.telephone = business.internationalPhoneNumber;
  }
  if (business.rating && business.userRatingCount) {
    schema.aggregateRating = {
      '@type': 'AggregateRating',
      ratingValue: business.rating,
      reviewCount: business.userRatingCount,
    };
  }
  const periods = business.regularOpeningHours?.periods;
  if (periods && periods.length > 0) {
    const days = ['Sunday', 'Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday'];
    schema.openingHoursSpecification = periods.filter(p => p.open).map(p => ({

      opens: `${String(p.open.hour).padStart(2, '0')}:${String(p.open.minute ?? 0).padStart(2, '0')}`,
      closes: p.close ? `${String(p.close.hour).padStart(2, '0')}:${String(p.close.minute ?? 0).padStart(2, '0')}` : undefined,

#!/usr/bin/env node
/**
 * Fetches nearby businesses from Google Places API New, dedupes by place_id,
 * applies block list, computes URL slugs, writes content/businesses-snapshot.json.
 *
 * Usage:
 *   GOOGLE_PLACES_API_KEY=... node scripts/fetch-businesses.js
 *   (or put the key in .env.local — Node 20+ doesn't auto-load it; use `node --env-file=.env.local`)
 */
import { writeFile } from 'node:fs/promises';
import { dirname, join } from 'node:path';
import { fileURLToPath } from 'node:url';
import { businessSlug } from './lib/slugify.js';

const __dirname = dirname(fileURLToPath(import.meta.url));
const ROOT = join(__dirname, '..');
const SNAPSHOT_PATH = join(ROOT, 'content/businesses-snapshot.json');

const API_KEY = process.env.GOOGLE_PLACES_API_KEY;
if (!API_KEY) {
  console.error('ERROR: GOOGLE_PLACES_API_KEY not set.');
  console.error('Set it in your environment or .env.local, then re-run with:');
  console.error('  node --env-file=.env.local scripts/fetch-businesses.js');
  process.exit(2);
}

const TOWN_LAT = 20.6748;
const TOWN_LNG = -102.5705;
const RADIUS = 5000; // meters

// Categories to fetch. Each is one nearbySearch call.
const CATEGORIES = [
  'restaurant', 'meal_takeaway', 'meal_delivery',
  'bar', 'cafe',
  'store', 'grocery_or_supermarket', 'pharmacy', 'bakery',
  'park', 'campground', 'tourist_attraction',
  'lodging',
];

// Place IDs to exclude (matches existing businesses.html BLOCKED_PLACE_IDS).
const BLOCKED_PLACE_IDS = new Set([
  'ChIJTSyIjZIzKYQRe3ZylqQUV1w', // PUNTO FIT
]);

// Field mask — only request fields we use, keeps cost down and respects API "essentials" tier.
const FIELD_MASK = [
  'places.id',
  'places.displayName',
  'places.primaryType',
  'places.types',
  'places.formattedAddress',
  'places.location',
  'places.internationalPhoneNumber',
  'places.regularOpeningHours',
  'places.rating',
  'places.userRatingCount',
].join(',');

async function fetchCategory(type) {
  const res = await fetch('https://places.googleapis.com/v1/places:searchNearby', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'X-Goog-Api-Key': API_KEY,
      'X-Goog-FieldMask': FIELD_MASK,
    },
    body: JSON.stringify({
      includedTypes: [type],
      maxResultCount: 20,
      locationRestriction: {
        circle: {
          center: { latitude: TOWN_LAT, longitude: TOWN_LNG },
          radius: RADIUS,
        },
      },
    }),
  });
  if (!res.ok) {
    const text = await res.text();
    throw new Error(`Places API ${type} returned ${res.status}: ${text}`);
  }
  const json = await res.json();
  return json.places || [];
}

function normalize(rawPlace) {
  // Places API New returns id as "places/ChIJ..."; we want just the place_id.
  const placeId = (rawPlace.id || '').replace(/^places\//, '');
  const displayName = rawPlace.displayName?.text || '';
  return {
    placeId,
    displayName,
    primaryType: rawPlace.primaryType,
    types: rawPlace.types || [],
    formattedAddress: rawPlace.formattedAddress,
    location: rawPlace.location,
    internationalPhoneNumber: rawPlace.internationalPhoneNumber,
    regularOpeningHours: rawPlace.regularOpeningHours,
    rating: rawPlace.rating,
    userRatingCount: rawPlace.userRatingCount,
    slug: businessSlug(displayName, placeId),
  };
}

async function main() {
  console.log(`Fetching ${CATEGORIES.length} categories...`);
  const seen = new Map(); // placeId → normalized business
  for (const type of CATEGORIES) {
    try {
      const raw = await fetchCategory(type);
      console.log(`  ${type}: ${raw.length} results`);
      for (const place of raw) {
        const b = normalize(place);
        if (!b.placeId) continue;
        if (BLOCKED_PLACE_IDS.has(b.placeId)) continue;
        if (!seen.has(b.placeId)) seen.set(b.placeId, b);
      }
    } catch (err) {
      console.error(`  ${type}: FAILED — ${err.message}`);
      // Continue with other categories — don't fail the whole snapshot for one bad category.
    }
  }
  const businesses = [...seen.values()].sort((a, b) =>
    (b.userRatingCount ?? 0) - (a.userRatingCount ?? 0)
  );

  const snapshot = {
    fetchedAt: new Date().toISOString(),
    townCenter: { latitude: TOWN_LAT, longitude: TOWN_LNG },
    radius: RADIUS,
    blockedCount: BLOCKED_PLACE_IDS.size,
    count: businesses.length,
    businesses,
  };

  await writeFile(SNAPSHOT_PATH, JSON.stringify(snapshot, null, 2) + '\n', 'utf8');
  console.log(`\n✓ Wrote ${businesses.length} businesses to ${SNAPSHOT_PATH}`);
}

main().catch(err => {
  console.error('Fetch failed:', err);
  process.exit(1);
});
